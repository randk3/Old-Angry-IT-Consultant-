# Old Angry IT Consulting AI Agent

## Project Title & Summary
Old Angry IT Consulting AI Agent is a responsive single-page web application for an IT consulting company. The project expands the original Old Angry IT Consulting website by adding an interactive AI-style consulting agent that helps users identify common business technology needs and receive practical recommendations for support, cybersecurity, systems performance, workflow improvement, and AI readiness.

## Purpose & Goals
The purpose of this project is to demonstrate advanced web technology through interactivity, modern JavaScript, responsive design, and an AI-style component. The project solves a realistic business problem by allowing users to describe technology challenges and receive guided recommendations and click directly to schedule a consultant session.

## Features
- Single-page application using JavaScript rendering
- Navigation between Home, Services, Solutions, Consultation Chat, and Contact sections
- Interactive AI-style consulting agent
- Quick prompt buttons for common business IT issues
- Custom text input with dynamic response logic
- Clickable scheduling link inside AI Agent responses
- Message timestamps
- Typing indicator animation
- Local chat history saved in the browser
- Clear chat option
- Responsive design for mobile, tablet, and desktop
- Bootstrap CDN for layout support
- PWA-ready files with manifest and service worker
- Professional IT consulting theme

## Setup Instructions
1. Download and unzip the project folder.
2. Open `index.html` in a browser to test the site locally.
3. Upload all project files to a GitHub repository.
4. Commit changes with clear messages.
5. Enable GitHub Pages from the repository settings.
6. Copy the live URL and submit it with the zipped project files.

## Deployment Plan
The project will be hosted on GitHub Pages. The repository will include `index.html`, `styles.css`, `script.js`, `README.md`, `manifest.json`, and `service-worker.js`. The site will be deployed from the main branch using the root folder.
